# Antam Gold Dashboard

Web dashboard interaktif untuk memantau harga harian **emas Antam** (logam mulia) dalam Rupiah Indonesia.

📊 **Data:** 4,960 hari (2010-01-04 → 2026-06-14) dari [`rohmats/antam-gold-price`](https://github.com/rohmats/antam-gold-price)  
🔄 **Update:** Auto-update harian via cron job + git push → Vercel auto-deploy  
⚡ **Hosting:** [Vercel](https://vercel.com) (static, 100% free tier)

## Features

- 📈 Chart interaktif (Chart.js) dengan dual-axis (1-50g & 100-1000g)
- 💰 Harga jual + buyback (harga beli balik)
- 🟢/🔴 Indikator perubahan (naik/turun)
- 📅 Rata-rata bulanan per tahun
- 📱 Responsive (mobile-friendly)
- 🌓 Dark theme
- 🇮🇩 Bahasa Indonesia

## Stack

- **Frontend:** Vanilla JS + Chart.js 4.x + Custom CSS
- **Data:** Pre-built JSON (columnar format) generated from `data.csv`
- **Hosting:** Vercel (static deployment)
- **No backend needed** — semua filtering dilakukan di browser

## Local Development

```bash
# 1. Generate JSON from CSV
python3 build_data.py

# 2. Serve locally (any static server works)
python3 -m http.server 8000
# Open http://localhost:8000
```

## Auto-Update Workflow

1. Cron job runs `fetch_history.py` (updates `data.csv` from rohmats repo)
2. Then runs `build_data.py` (regenerates `data/data.json`)
3. Then `git add` + `commit` + `push` to GitHub
4. Vercel auto-detects push, redeploys in ~30s
5. Users see updated data on next page load

## Cron Job

Currently scheduled: **daily 09:00 WIB**. See `~/.hermes/cron/` for job config.

## Data Source

Pulled from [`rohmats/antam-gold-price`](https://github.com/rohmats/antam-gold-price) — auto-updated 3x daily via GitHub Actions.
